# WeatherAPI – это iOS-приложение прогноза погоды. 

Разработанное на языке Swift с использованием UIKit и архитектурного паттерна MVVM, предназначенное для отображения текущей погоды, почасового прогноза на ближайшие 24 часа и прогноза на 7 дней. 

Особенности
1. Получение текущей геолокации пользователя (или города Москва по умолчанию)
2. Отображение текущей погоды, включая температуру и локацию
3. Почасовой прогноз на текущий и следующий день (в пределах ближайших 24 часов)
4. Прогноз погоды на 7 дней с возможностью вертикальной прокрутки
5. Обработка состояния загрузки и ошибок с возможностью повторного запроса
6. Минималистичный и отзывчивый интерфейс, реализованный с использованием UIKit

Архитектура

Приложение реализовано с применением архитектуры MVVM, где:
1. Model — отвечает за работу с API и обработку данных
2. ViewModel — связывает бизнес-логику с интерфейсом, преобразует данные модели в пригодный для отображения вид
3. View (UIViewController + UIKit-компоненты) — отображает данные и реагирует на действия пользователя

Код организован в отдельные модули с разделением по ответственности (UI, Networking, Services, Models, ViewModels и т.д.), каждый класс/структура имеют четко определённую роль.

Используемые технологии
1. Swift + UIKit
2. Архитектурный паттерн: MVVM
3. Работа с REST API через URLSession
4. Определение текущей геопозиции через CoreLocation
5. AutoLayout для адаптивной верстки интерфейса

1. Текущая погода - http://api.weatherapi.com/v1/current.json?key=fa8b3df74d4042b9aa7135114252304&q=LAT,LON
2. Прогноз на 7 дней - http://api.weatherapi.com/v1/forecast.json?key=fa8b3df74d4042b9aa7135114252304&q=LAT,LON&days=7
